## 하지만 웹 AI는 ‘해줘’에 약하다

"이거 해줘"

하는 방법 알려줄게:
1. 먼저 사이트에 들어가고
2. 버튼을 클릭한 다음
3. 설정을 확인하면 돼
4. 필요하면 다시 물어봐

## API key는 과금형 출입증이다

서비스가 호출할 때마다 키로 식별하고,
토큰을 세고,
사용량만큼 청구한다.

서비스 / 앱
→ API key
→ 누가 쓰는지 식별
→ TOKEN METER
→ 토큰 계산
→ 사용량 청구

쓴 만큼 과금

## OpenClaw: 컴퓨터 조작 레이어의 등장

대답만 하는 것이 아니라
브라우저, 파일, 터미널, 앱을 실제로 다루는 쪽으로 간다.

OpenClaw
24시간 대기하는 실행 레이어

명령을 받아 작업을 실행

브라우저 · 파일 · 터미널 조작

모델은 따로 연결해야 함
