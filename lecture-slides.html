2회기 1차시 - 디지털 정보 기초

파일 형식과 이름 정리 약속

Session 2
  • 이미지 파일: `.png` / `.jpg` 형식으로 저장 (예: 홍보 포스터)
  • 소리 파일: `.mp3` 형식으로 저장 (예: 배경 음악 BGM)
  • 비디오 파일: `.mp4` 형식으로 저장 (예: 인트로 무비)
  • 가장 흔히 하는 실수! 파일 이름에 한글, 특수문자, 띄어쓰기를 넣으면 나중에 웹사이트 배포할 때 링크가 깨져 엑스박스(X)가 뜹니다.
  • 파일명은 무조건 소문자 영어와 숫자로만 간단하게 지어주세요! (예: `poster.png`, `music.mp3`, `video.mp4`)

2회기 2차시 - 디지털 갤러리 & 미디어 설계

미니게임 속에 미디어(BGM·이미지·영상) 녹여내기

Session 2
  • 로딩 화면 (포스터): 게임 시작 전 등장하는 대기 화면에 AI로 그린 대표 포스터 이미지(`poster.png`)를 띄워줍니다.
  • 배경음악 (BGM): 게임이 재생되는 동안 신나는 배경 음악 파일(`music.mp3`)이 무한 반복(`loop`)으로 재생되도록 설정합니다.
  • 승리 축하 영상 (보너스): 게임 클리어(성공) 시 팝업과 함께 보너스 축하 동영상(`video.mp4`)이 멋지게 재생되도록 코딩합니다.
  • ChatGPT에게 "대기 화면 포스터, 루프 배경음악, 게임 클리어 시 재생되는 보너스 동영상을 포함한 코드를 짜줘"라고 상세히 요구하세요!

게임 내 미디어 코드 조각

<!-- 게임 배경음악 무한 루프 재생 -->
<audio src="music.mp3" autoplay loop></audio>

<!-- 로딩 포스터 화면 -->
<img src="poster.png" id="start-poster">

<!-- 게임 승리 보너스 비디오 -->
<video src="video.mp4" id="win-video"></video>
